# RepoPilot 🚀

An autonomous DevOps CLI sub-agent built for the OpenAI 2026 Build Week. RepoPilot automatically scans messy codebases, generates custom environment setup commands via Codex, and executes them with a self-healing error pipeline.

## 🛠️ Tech Stack

- Python
- OpenAI Codex API (GPT-4o-mini)

## 🚀 Setup Instructions

1. Clone this repository.
2. Initialize environment variables:
   ```cmd
   set OPENAI_API_KEY=your_key_here
   ```
3. Run the onboarding command:
   ```cmd
   python repopilot.py
   ```
